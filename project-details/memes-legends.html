    Memes & Legends


    Introduction

    "Memes & Legends" is an exciting multiplayer turn-based board game designed for desktop platforms using Unity. This game offers a unique and engaging experience where players take control of four distinct characters, each with their own attack and unique ability. With the power to evolve these characters and an innovative approach to character data storage using scriptable objects.


    Key Features

    • Multiplayer Experience: "Memes & Legends" offers an immersive multiplayer environment where players can engage in thrilling battles with opponents. Turn-based gameplay ensures that each move is strategic and thought-provoking.
    • Unique Characters: In this game, players take control of four unique characters, each possessing their own attack and special ability. The diverse abilities of these characters add depth to the gameplay, allowing players to create winning strategies.
    • Character Evolution: One of the game's defining features is the ability to evolve characters. As players progress, they can enhance their characters' capabilities, making them even more formidable in battle.
    • Scriptable Object Data: Character data in "Memes & Legends" is stored using scriptable objects, offering an efficient and organized method for managing character attributes. This system allows for easy updates and adjustments to characters and their abilities.
    • Photon SDK for Multiplayer: The game's multiplayer functionality is powered by the Photon SDK, ensuring smooth and synchronized gameplay experiences for players, even in a turn-based setting.

    Technology Stack

    • Unity 2D
    • Photon Multiplayer
    • Scriptable Objects
